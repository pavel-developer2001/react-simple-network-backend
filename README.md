# Бэкенд для react-simple-network

## Версия 0.2.0

## В разработке применяется:

- Node js
- Express
- Sequelize, PostgreSQL

## Установка

- 1 Надо установить все пакеты:

```bash
npm i
# or
yarn install
```

- 2 Запускаем приложение:

```bash
npm run dev
# or
yarn dev
```
